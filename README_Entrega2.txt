
Entrega 2 - Navega las Rutas (React Router)
-------------------------------------------
1) Instalar dependencia:
   npm i react-router-dom

2) Copiar el contenido de la carpeta /src a tu proyecto (reemplazar o fusionar con tu /src).

3) Correr:
   npm run dev

Estructura incluida:
- src/main.jsx
- src/App.jsx
- src/styles.css
- src/data/api.js
- src/components/NavBar.jsx
- src/components/ProductCard.jsx
- src/components/ItemCount.jsx
- src/pages/ItemListContainer.jsx
- src/pages/ItemDetailContainer.jsx
- src/pages/NotFound.jsx
